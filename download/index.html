Download and Install

The recommended way to install WYMeditor is from one of the packaged builds

Packaged Builds

The currently stable version is 1.0.0b1 or you can use the latest beta version.

Extract the zip folder inside your project (we recommend the wymeditor directory).

Include the compressed WYMeditor source in your HTML (along with jquery):

WYMeditor should work with jQuery 1.2.x and up, although using a later jQuery version is preferred.

From Source

For the newest and shiniest features and bug fixes, try WYMeditor directly from source via GitHub. Checkout the README file for instructions on building a minified version, or include the javascript source files directly (you can use the test files as an example.
